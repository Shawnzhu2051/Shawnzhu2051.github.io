GAN 的数学推导

2018-03-08

最近将研究方向转移到了很火热很有趣的 生成式对抗网络 (Generative Adversarial Nets, GAN)上面来了。GAN是Goodfellow于2014年提出的一项基于博弈论和minimax理论的对抗性生成模型。这篇文章在老板的督促下，翻来覆去起码看了无数遍，所有的公式也手工推导了两到三遍，算得上是小有一些体会。

关于GAN的原理上的阐述，在网上的各种帖子博客里已经说得很详细了。这篇文章主要针对于GAN中比较关键的几个数学公式，予以证明。

Introduction

首先给出GAN的目标函数：

min⁡Gmax⁡DV(D,G)=Ex∼pdata(x)[log⁡D(x)]+Ez∼pz(z)[log⁡(1−D(G(z)))]\min_{G}\max_{D} V(D,G) = \mathbb{E}_{x \sim p_{data}(x)}[\log{D(x)}] + \mathbb{E}_{z \sim p_{z}(z)}[\log(1-D(G(z)))]minG​maxD​V(D,G)=Ex∼pdata​(x)​[logD(x)]+Ez∼pz​(z)​[log(1−D(G(z)))]

其中，pdata(x)p_{data}(x)pdata​(x) 是真实数据的数据分布，pz(z)p_{z}(z)pz​(z)是先验的噪音数据分布，pg(x)p_{g}(x)pg​(x)是生成数据的数据分布，D(⋅)D(\cdot)D(⋅)表示判别器认为这个输入数据为真实数据的概率。这个目标函数想要达到的目标是使生成数据的数据分布pg(x)p_{g}(x)pg​(x)和真实数据的数据分布pdata(x)p_{data}(x)pdata​(x)相同，即生成器可以生成以假乱真的图像。为此，我们需要最优化生成器GGG和判别器DDD，即：

DG∗=argmax⁡DV(G,D)D^{*}_{G} = \text{arg}\max_{D}V(G,D)DG∗​=argmaxD​V(G,D)和G∗=argmin⁡GV(G,DG∗)G^{*} = \text{arg}\min_{G}V(G,D^{*}_{G})G∗=argminG​V(G,DG∗​)

更新算法如下：

Q1：如何衡量pg(x)p_{g}(x)pg​(x) 和pdata(x)p_{data}(x)pdata​(x)的相近程度？

为了论证pg(x)=pdata(x)p_{g}(x) = p_{data}(x)pg​(x)=pdata​(x)，我们需要设置一个衡量两个数据分布相似程度的指标。在文中，Goodfellow引入了KL散度：

DKL(P∣∣Q)=Ex∼p[log⁡P(x)Q(x)]=Ex∼p[log⁡P(x)−log⁡Q(x)]D_{KL}(P||Q) = \mathbb{E}_{x \sim p}[\log{\frac{P(x)}{Q(x)}}] = \mathbb{E}_{x \sim p}[\log{P(x)}-\log{Q(x)}]DKL​(P∣∣Q)=Ex∼p​[logQ(x)P(x)​]=Ex∼p​[logP(x)−logQ(x)].

为什么KL散度可以衡量pg(x)p_{g}(x)pg​(x) 和pdata(x)p_{data}(x)pdata​(x)的相近程度呢？下面来证明。

我们从Pdata(x)P_{data}(x)Pdata​(x)中抽取mmm个样本x1,x2,...,xm{x^{1},}x^{2},...,x^{m}x1,x2,...,xm,设分布的参数为θ\thetaθ，则每个样本可以表示为pg(x;θ)p_{g}(x;\theta)pg​(x;θ)，得到似然函数：

L=∏i=1mpg(xi;θ)L = \prod^{m}_{i=1}p_{g}(x^{i};\theta)L=∏i=1m​pg​(xi;θ)。

该似然函数的含义是：在由θ\thetaθ确定的生成分布中，mmm个样本全部出现的概率。

接下来我们将最大化似然函数来得到最优分布θ∗\theta^{*}θ∗：

θ∗=argmax⁡θ∏i=1mpg(xi;θ)\theta^{*} = \text{arg}\max_{\theta}\prod^{m}_{i=1}p_{g}(x^{i};\theta)θ∗=argmaxθ​∏i=1m​pg​(xi;θ)

因为log⁡(⋅)\log(\cdot)log(⋅)函数是单调的，所以为去掉连乘，将似然函数化为

\theta^{*} = \text{arg}\max_{\theta}\log{\prod^{m}_{i=1}p_{g}(x^{i};\theta)} \\= \text{arg}\max_{\theta}\sum^{m}_{i=1}log{p_{g}(x^{i};\theta)} \\ = \text{arg}\min_{\theta}\mathbb{E}_{x \sim p_{data}}[\log{p_{g}(x;\theta)}]

（因为xi∈pdata(x)x^{i} \in p_{data}(x)xi∈pdata​(x)）

添加一项不含θ\thetaθ的项，并展开：

\theta^{*} = \text{arg}\max_{\theta}\int_{x}p_{data}(x)\log{p_{g}(x;\theta)}dx - \int_{x}p_{data}(x)\log{p_{data}(x)}dx \\= \text{arg}\max_{\theta}\int_{x}p_{data}(x)\log{\frac{p_{g}(x;\theta)}{p_{data}(x)}} dx \\ = \text{arg}\max_{\theta} KL(p_{g}(x;\theta)|| p_{data}(x))

由此证明了KL散度能够衡量pg(x)p_{g}(x)pg​(x) 和pdata(x)p_{data}(x)pdata​(x)的相近程度。

Q2：如何证明判别器DDD存在最优解？最优解的值为多少？

展开GAN的目标函数中数学期望部分可以得到：

V(G,D)=∫xpdata(x)log⁡D(x)dx+∫zpz(z)log⁡(1−D(G(z))dzV(G,D) = \int_{x}p_{data}(x)\log{D(x)}dx + \int_{z}p_{z}(z)\log{(1-D(G(z))}dzV(G,D)=∫x​pdata​(x)logD(x)dx+∫z​pz​(z)log(1−D(G(z))dz.

当给定一个GGG时，因为GGG输入的是噪音数据zzz，得到的是生成的数据xxx，所以使用换元积分，令G(z)=xG(z) = xG(z)=x。值得注意的是，该处证明存在一个漏洞，即要使用换元积分的话，函数G(x)G(x)G(x)必须是可逆的，然而GGG是一个神经网络，目前尚不能证明其实可逆的。但该种换元方法在机器学习中非常常见，所以就暂时忽略掉。

$\int_{x}p_{data}(x)\log{D(x)}dx + \int_{x}p_{g}(x)log{(1-D(x))}dxV(G,D) \= \int_{x}\left[p_{data}(x)\log{D(x)} + p_{g}(x)log{(1-D(x))}\right]dx $

因为要求目标函数VVV的最大值，而GGG是固定的，所以最大值只和DDD有关。为方便起见，令pdata(x)=a,pg(x)=b,D(x)=y,V(⋅)=f(⋅)p_{data}(x) = a, p_{g}(x) = b, D(x) = y, V(\cdot) = f(\cdot)pdata​(x)=a,pg​(x)=b,D(x)=y,V(⋅)=f(⋅) .

则f(y)=alog⁡y+blog⁡(1−y)f(y) = a \log{y} + b \log{(1-y)}f(y)=alogy+blog(1−y)

f′(y)=ay−b1−y=0f^{'}(y) = \frac{a}{y} - \frac{b}{1-y} = 0f′(y)=ya​−1−yb​=0

y=aa+by = \frac{a}{a+b}y=a+ba​

二阶导数：f′′(y)=−ay2−b1−y2f^{''}(y) = -\frac{a}{y^{2}}-\frac{b}{1-y^{2}}f′′(y)=−y2a​−1−y2b​

f′′(aa+b)<0,(a,b∈(0,1))f^{''}(\frac{a}{a+b}) \lt 0 ,(a,b \in (0,1))f′′(a+ba​)<0,(a,b∈(0,1)).

所以证明了y=aa+by = \frac{a}{a+b}y=a+ba​为极大值且唯一，即DG∗(x)=pdata(x)pdata(x)+pg(x)D^{*}_{G}(x) = \frac{p_{data}(x)}{p_{data}(x) + p_{g}(x)}DG∗​(x)=pdata​(x)+pg​(x)pdata​(x)​

如果最后GAN能够达到最优，即pg(x)=pdata(x)p_{g}(x) = p_{data}(x)pg​(x)=pdata​(x)，则判别器DG∗(x)=12D^{*}_{G}(x) = \frac{1}{2}DG∗​(x)=21​！

也就是说判别器最后对输入数据，不论是真实数据还是生成数据，判定其为真的概率均为12\frac{1}{2}21​，也就是说随机猜测无法判断。那么我们可以认为这个生成数据已经能够以假乱真，让判别器无法判断了。

Q3：如何证明当且仅当pg(x)=pdata(x)p_{g}(x) = p_{data}(x)pg​(x)=pdata​(x)时，生成器GGG有最优解？

这个当且仅当问题需要从两个方向来证明。

我们先假设当判别器DDD训练好时，固定DDD，令C(G)=V(DG∗,G)C(G) = V(D^{*}_{G},G)C(G)=V(DG∗​,G)，再令pg(x)=pdata(x)p_{g}(x) = p_{data}(x)pg​(x)=pdata​(x)来求C(G)C(G)C(G)的一个可能的取值，再证明它是最优解：

C(G) = \int_{x}p_{data}(x)\log{\frac{1}{2}} + p_{g}(x)\log{\frac{1}{2}} dx \\=-\log{2}\int_{x}p_{data}(x)dx - \log{2}\int_{x}p_{g}(x)dx \\ =-log{4}.

接下来对所有的GGG，将最优判别器DG∗D^{*}_{G}DG∗​代入到C(G)C(G)C(G)中，来求C(G)C(G)C(G)的最优解：

C(G) = \int_{x} p_{data}(x)\log{\frac{p_{data}(x)}{p_{g}(x)+p_{data}(x)}} + p_{g}(x)\log{\frac{p_{g}(x)}{p_{g}(x)+p_{data}(x)}}dx \\= \int_{x}(\log{2} - \log{2})p_{data}(x) + p_{data}(x)\log{\frac{p_{data}(x)}{p_{g}(x)+p_{data}(x)}} + (\log{2} - \log{2})p_{g}(x) + p_{g}(x)\log{\frac{p_{g}(x)}{p_{g}(x)+p_{data}(x)}}dx

我们在这里使用了一个小技巧，添加了两项值为0的项，并展开，对原式的结果没有影响。

C(G) = -\log{2}\int_{x}[p_{g}(x) + p_{data}(x)]dx + \int_{x}\left[p_{data}(x)\left(\log{2} + \log{\frac{p_{data}(x)}{p_{g}(x)+p_{data}(x)}}\right) + p_{g}(x)\left(\log{2}+\log{\frac{p_{g}(x)}{p_{g}(x)+p_{data}(x)}} \right)\right]dx \\= -\log{4} + \int_{x}\left[p_{data}(x)\left(\log{\frac{p_{data}(x)}{\frac{p_{g}(x)+p_{data}(x)}{2}}}\right) + p_{g}(x)\left(\log{\frac{p_{g}(x)}{\frac{p_{g}(x)+p_{data}(x)}{2}}} \right)\right]dx \\= -\log{4} + KL(p_{data}||\frac{p_{data}+p_{g}}{2}) + KL(p_{g}||\frac{p_{data}+p_{g}}{2}).

因为KL散度是恒大于零的，所以−log⁡4-\log{4}−log4为C(G)C(G)C(G)的最小值。

接下来再证明当生成器GGG取得最优解−log⁡4-\log{4}−log4时，pg(x)=pdata(x)p_{g}(x) = p_{data}(x)pg​(x)=pdata​(x)。

因为KL散度不可以交换，所以文中引入了JS散度。JS散度的定义如下。

JSD(P∣∣Q)=12KL(P∣∣M)+12KL(Q∣∣M)JSD(P||Q) = \frac{1}{2}KL(P||M) + \frac{1}{2}KL(Q||M)JSD(P∣∣Q)=21​KL(P∣∣M)+21​KL(Q∣∣M)，其中M=12(P+Q)M = \frac{1}{2}(P+Q)M=21​(P+Q)，JSD∈[0,log⁡2]JSD \in [0, \log{2}]JSD∈[0,log2].

所以C(G)=−log⁡4+2⋅JSD(pdata∣∣pg)C(G) = -\log{4} + 2\cdot JSD(p_{data}||p_{g})C(G)=−log4+2⋅JSD(pdata​∣∣pg​).

当pdata(x)=pg(x)p_{data}(x) = p_{g}(x)pdata​(x)=pg​(x)时，JSD(pdata∣∣pg)=0JSD(p_{data}||p_{g}) = 0JSD(pdata​∣∣pg​)=0。

所以当生成器GGG取得最优解−log⁡4-\log{4}−log4时，pg(x)=pdata(x)p_{g}(x) = p_{data}(x)pg​(x)=pdata​(x)。

Summary

  • GAN结合了生成模型和判别模型，消除了生成模型的loss function难以定义的问题。
  • 基于概率分布来计算，不受生成维度的限制。
  • 可以用来进行半监督学习（先使用无标签的数据训练GAN，在基于GAN使用有标签的数据训练判别器）。
  • 论文有严格的数学证明。
