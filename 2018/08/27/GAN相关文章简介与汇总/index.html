GAN相关文章简介与汇总

2018-08-27 机器学习

之前在面试商汤的时候，自以为读了几篇论文后就对GAN很了解，没想到被面试官问傻了。基本上GAN比较关键的文章的细节全部都被问到了。所以之后再准备面试的时候，就对GAN的几类文章：

1，有重要的新结构；2，跟我的文章相关；3，发展方向和有趣的用法；

做了一个小小的走马观花式的总结。主要参考了Github上的这个库：The GAN Zoo。

CGAN

论文地址： Conditional Generative Adversarial Nets

特点

  • 加入了监督信息，无监督变有监督
  • 带有条件概率的two-player minimax game

架构

DCGAN

GAN与CNN的结合。

论文地址：Unsupervised Representation Learning with Deep Convolutional Generative Adversarial Networks

特点

  • 对CNN的网络结构做了限制，使其可以稳定训练
  • 展示了生成特征的计算向量特性

架构

  • 将pooling转变为convolution

  • 使用BN

  • 移除全连接层

  • 使用了ReLU

WGAN

论文地址：Wasserstein GAN

特点

  • 判别器最后一层去掉 sigmoid
  • 生成器和判别器的 loss 不取 log
  • 每次更新判别器的参数之后把它们的绝对值截断到不超过一个固定常数 c
  • 不要用基于动量的优化算法（包括 momentum 和 Adam），推荐 RMSProp，SGD 也行
  • JS散度替代KL散度

LAPGAN

使用拉普拉斯金字塔实现逐级对抗。

论文地址：Deep Generative Image Models using a ￼Laplacian Pyramid of Adversarial Networks

特点

  • 与Residual Network有异曲同工之妙，针对残差的逼近和学习相对容易
  • 逐级独立训练提高了网络简单记忆输入样本的难度
  • 减少了每一次 GAN 需要学习的内容

架构

  • 将高层金字塔图像也作为输出，和噪音一起生成低层的图像，相当于一个CGAN

ALI

隐空间，联合对抗

论文地址：Adversarially Learned Inference

特点

  • 使用对抗的方式联合训练了一个生成网络与一个推断网络
  • 生成网络就是 GAN 中的 G, 将随机潜变量映射到数据空间, 而推理网络 (以下简称 I) 将训练样本从数据空间反映射到潜变量空间
  • 对抗博弈在 G, I 与 D 之间展开

架构

Image2Image

根据抽象图像生成真实图像

论文地址：Image-to-Image Translation with Conditional Adversarial Networks

特点

  • 对CGAN进行了3个改进
  • 目标函数加入了L1约束，类似于AAE
  • 生成器使用了U-Net
  • 提出了分块判断

架构

  • U-Net：

InfoGAN

论文地址： InfoGAN: Interpretable Representation Learning by Information Maximizing Generative Adversarial Nets

特点

  • 通过可分解的特征学习得到可解释的特征
  • GAN的问题：无法得知生成特征的对应噪声数据
  • 加入了一个潜变量c，使得c与生成的样本具有较高的互信息量，c可以表示语义特征

架构

SRGAN

论文地址：Photo-Realistic Single Image Super-Resolution Using a Generative Adversarial Network

特点

  • GAN以前loss function都是基于MSE的，提出了感知相似性来代替PSNR衡量
  • loss function，将像素损失替换为content loss，
  • loss function 加入了adversarial loss

架构

PG-GAN

英伟达最新研究成果，用于生成高精度图像。

论文地址：Progressive Growing of GANs for Improved Quality, Stability, and Variation

特点

  • 可以生成1024*1024的样本
  • 逐级训练，增加细节

架构

  • 加入了fade in 平滑过渡，利用了ResNet的思想
  • 使用了WGAN

CycleGAN

论文地址：Unpaired Image-to-Image Translation using Cycle-Consistent Adversarial Networks

特点

  • 风格迁移 image translation
  • 通过两个镜像对称的GAN构成了一个环形网络
  • 每个单向GAN中包含了两个生成器，一个生成对应domain中的图片，另一个将前一个的结果还原回原始domain并使用L2 loss。判别器使用对应domain的样本进行判断

架构
