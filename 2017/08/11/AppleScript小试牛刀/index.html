AppleScript小试牛刀

Veröffentlicht am 2017-08-11 | in 其他开发

这篇文章的想法最初是在知乎上看到一个很有趣的帖子，讲述了一个程序员男友用Apple脚本语言给异地的女朋友写的一个可以类似文本对话游戏的小程序，于是大受启发：“哇原来程序员还可以这样撩妹！”。二话不说，马上开工。

基础知识

AppleScript是MacOS系统原生的脚本语言（其实就是bash，AppleScript只是一个装逼的名字），最初开发它的目的是为了将Mac上的一些繁琐的操作自动化。比如一键搜索电脑上某些文件并将其移入回收站然后清空，或者是创建一些表单之类的。整个AppleScript的语法非常的简单，几乎就只需要初级的一些编程知识，包括if语句，for语句和接口的调用就基本足够了。当然如果想要更强大的功能，还是得花更多的时间和精力在这儿上面。

AppleScript使用的编辑器是MacOS自带的脚本编辑器。

基本语法

以下使用几段代码来说明AppleScript的语法是怎么样的。

对话框

display dialog "这是两位爸爸送给儿子的生日礼物" buttons {"OK", "取消"} default button 1
    if button returned of result is "OK" then
        display dialog "桌面太乱了，帮你清理一下吧" buttons {"哦？？？"} default button 1
        ...
        ...
    end if

上述代码表示新建一个对话框，对话框的内容为“这是两位爸爸送给儿子的生日礼物”并显示，这个对话框具有两个按钮，分别是“OK”和“取消”，其中系统默认选择“OK”按钮，效果如下：

然后利用if语句，就可以实现在选择不同按钮时返回不同的结果。除此之外，和可以给对话框加上标题，icon等组件，实现更多的效果。这些
东西就请自行百度，在这里就暂不赘述了。

发声

beep
say "Happy Birthday to you" using "Victoria"

上述代码中，beep表示让系统发出警报音，也就是“嘟”的一声。而say语句会调用系统自带的语音系统，让系统使用Victoria的声音说出“Happy Birthday to you”这句话。Victoria是系统的默认语音，如果想使用其他语音的话，在设置-辅助功能-语音一栏中，可以自行下载更多的系统语音。

我在网上还看到一个更加高级的语音设置：

AppleScript 脚本让 Mac 唱生日快乐歌

通过设置音调让系统真正实现“唱”出来，而不是“念”出来。

bash操作

在这里只列举两个简单的操作。

do shell script "defaults write com.apple.finder CreateDesktop -bool FALSE;killall Finder"

上述语句的作用是，将桌面所有文件的可见性设置为隐藏。效果等同于直接在命令行中输入

defaults write com.apple.finder CreateDesktop -bool FALSE;killall Finder

如果想要设置成将其他文件夹中文件隐藏的话，修改语句的作用范围就行了。下面这条语句是重新将所有桌面文件设置为可见。

do shell script "defaults delete com.apple.finder CreateDesktop;killall Finder"

另一个简单的操作是清空回收站：

do shell script "cd ~/.Trash;rm -rf *"
do shell script "killall Finder"

没有什么可以多说的，熟悉bash的人都可以看出来，只是使用了一个脚本来运行shell语句而已。
