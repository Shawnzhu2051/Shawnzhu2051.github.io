基于朴素贝叶斯算法的推特数据新闻检测

2017-03-08 机器学习

寒假的时候做了一个项目：利用朴素贝叶斯算法对推特数据进行分类。个人感觉很有趣，可以算作是数据科学的一个入门练手项目。

基本的条件和要求如下：

需要将推特数据分为新闻和非新闻两类
训练样本是已经给定的从网络上爬取的推特数据，并且已经标记好了属性，如下

[u’Thu Jan 14 01:54:14 +0000 2016’, u’Officials say a Ukrainian man has died along with a 14-year-old French girl in Alps avalanche https://t.co/le0iN4dgzu‘, []]

[u’Thu Jan 14 01:54:59 +0000 2016’, u’French Alps Avalanche Hits Secondary School Party In Isere, Three Dead.. https://t.co/93PnskM2Al https://t.co/EQl55NUJuB‘, []]

格式相同的测试样本给出，并已经标记好了属性，以便测试查准率（Recall）
需要使用训练样本构建一个朴素贝叶斯的分类器，使其能够对输入的测试样本进行分类
基本流程如下：

首先对两个输入的训练样本进行清洗，将每条推特的的text部分提取到list里，然后按照空格对list里的每一项item进行划分，得到每一个分词。之后再统一大小写，去除text后的网址，@信息，#（话题）后的词语要作为普通的分词加入text，接下来去掉分词组中的语义无关词（比如字母长度小于2的词，are and will the this that等）

for line in List:

引用python中的Counter库，将清洗好的数据进行统计，得到每一个词在总词库中出现的频率，并将其频率作为该词出现的概率。再将所有词按照出现概率大小排列成dict输出

i = 0
table = []
for line in list:
    for word in line:
        table.append(word)
        i = i + 1
        counter = Counter(table)
        times = counter.most_common(i)
return times

使用统计好的结果对贝叶斯模型进行计算。即将每个在样本中出现过的词作为样本的一个属性，计算其条件概率。这一个过程即称为训练，因为在这两个不同的样本中，存在着不相同的特征，如果按照普通的贝叶斯算法的话，概率将变为0，因此需要加入平滑系数，此处采用的是拉普拉斯平滑。

对两个频次表中的相同项,获取其频率作为先验概率,对于不相同的项目,加入拉普拉斯平滑系数

for (key1,value1) in table1:
    for (key2,value2) in table2:
        if key1 == key2:
            Conditional_probability[key1] = value2 + 1/news_number + 2
            flag = 1
        if flag == 1:
            flag = 0
        else:
            Conditional_probability[key1] = 1/(news_number+2)
return Conditional_probability

将测试样本清洗后输入模型，对测试样本里的每一条推特，在训练器中查找其对应的条件概率，若该属性缺失则将概率记为1，最后将算出来的所有条件概率之积进行比较。值得注意的是，因为所有条件概率之积算出来极小（大约为10e-324），计算机无法进行除法来比较大小，所以每计算一个条件概率就进行一次分类，比较该条推特是新闻的概率和是非新闻的概率哪个大，并进行记录和统计。
